# Student Placement Prediction Analysis

## 📌 Project Overview
This project analyzes a student placement dataset to identify key factors that influence whether a student gets placed and their salary package. The goal is to derive insights that can help students improve their chances of placement.

---

## 📊 Dataset Information
- **Dataset Name:** Student Placement Prediction Dataset
- **Source:** Kaggle
- **Rows:** 100,000
- **Columns:** 26

### Key Features:
- CGPA
- Internship Experience
- Coding Skills
- Communication Skills
- Placement Status
- Salary Package (LPA)

---

## 🛠 Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab / Jupyter Notebook

---

## 🔍 Project Steps

### 1. Data Cleaning
- Handled missing values using mean
- Removed duplicate records

### 2. Statistical Analysis
- Calculated mean, median, mode
- Standard deviation, variance
- Range and mid-range

### 3. Data Visualization
- Histogram (CGPA distribution)
- Countplot (Placement status)
- Boxplot (Salary distribution)
- Heatmap (Correlation analysis)

### 4. Predictive Modeling
- Applied encoding on categorical variables
- Split dataset (80:20)
- Built Logistic Regression model
- Evaluated using accuracy score

---

## 📈 Key Insights
- Higher CGPA increases placement chances
- Internship and coding skills positively impact placement
- Salary distribution shows presence of high-value outliers

---

## 💡 Recommendations
- Students should focus on improving technical and communication skills
- Colleges should provide more internship opportunities
